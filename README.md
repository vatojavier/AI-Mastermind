# AI-Mastermind
To execute the program:

    python3 masterMind.py

To test the game the options are -c:

   Example:

        python3 masterMind.py

Random goal code will be generated

   Default:

        python3 masterMind.py -c random

Random goal code will be generated


   Manually gen code:

     python3 masterMind.py -c me
